DOS命令

纸灰2022/10/25...大约 1 分钟电脑系统WindowsDOS命令Windows命令

常用命令
sysdm.cpl 快速打开环境变量
shutdown -s -t 60 指定时间关机
shutdown -r -t 重新启动
shutdown -a 取消关机
cd 进入文件夹
md 创建目录
dir 显示文件
dir /a 显示所有文件(包括隐藏文件)
move 移动
copy 源文件 复制到的目录 复制
xcopy /y .\js .\dist\js /s/e 复制目录和目录下的文件
rd 删除空目录
cd.. 退回根目录
del 删除文件
exit 退出dos
cls 清屏
ipconfig 查看ip地址
echo 文本
dir /s/a
tree
arp -a 查看局域网所有的IP
curl -i -k 请求网址
-k 使用SSL时允许不安全的服务器连接
-i 在输出中包含协议响应头
ping 地址 检查网络是否连通
ping 地址 -t 一直ping
tcping 地址 端口 检查端口是否连通(C:\Windows\System32\tcping.exe)需下载tcping
netsh advfirewall set allprofiles state on 启动所有防火墙
netsh advfirewall set allprofiles state off 关闭所有防火墙
netstat -ano 查看所有端口
netstat -nao | findstr xxx 查找指定端口
taskkill -pid xxx -f 关闭指定端口 xxx用pid
tasklist| findstr "mysql" 查询mysql
taskkill/f /t /im mysqld.exe 关闭mysql


强制删除文件和文件夹
删除文件bat
把文件拖到bat上
del.bat

 DEL /F /A /Q \\?\%1 RD /S /Q \\?\%1

删除文件夹bat
把文件夹拖到bat上
rd.bat

 RD /S /Q \\?\%1

rem TASKKILL /F /IM wechat.exe
